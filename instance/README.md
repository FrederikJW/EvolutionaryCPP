# Problem Instances
Problem Instances were taken from https://github.com/MMSorensen/CP-Lib which are presented in the publication:
M. M. Sørensen and A. N. Letchford, “CP-Lib: Benchmark Instances of the Clique Partitioning Problem,” Mathematical Programming Computation, vol. 16, no. 1, pp. 93–111, Nov. 2023. DOI: 10.1007/s12532-023-00249-1.
